# Andina Campo Digital

App de campo para Andina Consultora Socioambiental.

## Cómo subir a Vercel

1. Subí esta carpeta a GitHub:
   - Abrí github.com
   - Creá un repositorio nuevo llamado `andina-campo-digital`
   - Subí todos los archivos

2. Conectá con Vercel:
   - Abrí vercel.com
   - "Add New Project"
   - Elegí el repositorio `andina-campo-digital`
   - Click en "Deploy"

3. Listo — en 2 minutos tenés el link.
